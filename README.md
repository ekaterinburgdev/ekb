## Installation

```
pnpm i ekb

Add ekb/fonts.css
```

## Local server

```
pnpm start
```

## Local linking development

If you want to test this library without tag publishing you can link the package:

```
pnpm build

pnpm link --global
cd OTHER_PROJECT_WITH_THIS_PACKAGE
pnpm link --global ekb
```
